## 🔧 如果无法访问

### 问题1：页面空白
**解决方法**：
1. 按 `Ctrl+Shift+R` (Windows) 或 `Cmd+Shift+R` (Mac) 强制刷新
2. 清除浏览器缓存
3. 尝试无痕模式

### 问题2：连接被拒绝
**解决方法**：
1. 确认终端中的端口号（可能不是5174）
2. 检查防火墙设置
3. 重启开发服务器：
   ```bash
   cd /Users/sady3721/project/vue_radar
   npm run dev
   ```

### 问题3：页面有错误
**解决方法**：
1. 按 `F12` 打开开发者工具
2. 查看 Console 标签页
3. 截图错误信息

## 📞 快速诊断

### 1. 检查服务器状态
在终端运行：
```bash
curl http://localhost:5174
```

如果看到 HTML 内容，说明服务器正常。

### 2. 检查端口占用
```bash
lsof -ti:5174
```

如果有输出，说明端口被占用。

### 3. 查看实时日志
查看终端窗口，Vite会显示：
- 编译错误
- 网络请求
- 热更新状态

## 💡 小技巧

### 快捷键
- `Ctrl+Shift+R`: 强制刷新
- `F12`: 打开开发者工具
- `Ctrl+Shift+I`: 打开开发者工具（备选）

### 调试模式
在浏览器控制台输入：
```javascript
console.log('Vue Radar loaded:', window)
```

### 端口更改
如果5174被占用，编辑 `vite.config.ts`:
```typescript
export default {
  server: {
    port: 3000  // 改为其他端口
  }
}
```
