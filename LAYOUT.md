# 雷达系统布局说明

## 整体布局

横向三块布局，从左到右：

```
┌──────────┬───┬──────────┬───┬─────────┐
│          │   │          │   │         │
│  Canvas  │ S │ Waveform │ S │ Toolbar │
│  620x520 │ 1 │  620x520 │ 2 │ 200x520 │
│          │ 0 │          │ 0 │         │
│          │   │          │   │         │
└──────────┴───┴──────────┴───┴─────────┘
```

## 各部分尺寸

### 1. 左侧：雷达画布 (Canvas)
- **尺寸**: 620px × 520px
- **功能**: 雷达视图、设备布局、家具放置
- **背景色**: 白色
- **边框**: 1px solid #ccc

### 2. 第一个分隔器 (Spacer 1)
- **尺寸**: 10px × 520px
- **功能**: 视觉分隔
- **背景色**: #e0e0e0
- **交互**: 无点击事件

### 3. 中间：示波器 (Waveform Monitor)
- **尺寸**: 620px × 520px
- **功能**: 显示4组生命体征波形
  - 心率 (Heart Rate)
  - 呼吸 (Breathing)
  - 睡眠状态 (Sleep State)
  - 体动 (Movement)
- **布局**:
  - 标题栏: 40px 高
  - 每个波形: ~100px 高
  - 总共可容纳 4 组波形
- **背景色**: 白色
- **边框**: 1px solid #ccc

### 4. 第二个分隔器 (Spacer 2)
- **尺寸**: 10px × 520px
- **功能**: 视觉分隔 + 折叠/展开工具栏
- **背景色**: #e0e0e0
- **交互**: 点击切换工具栏显示/隐藏
- **图标**: 《 / 》

### 5. 右侧：工具栏 (Toolbar)
- **尺寸**: 200px × 520px
- **功能**: 控制面板
  - 设备管理
  - 家具管理
  - 绘图工具
- **背景色**: #f8f8f8
- **边框**: 1px solid #ccc
- **折叠**: 可通过第二个分隔器折叠/展开

## 总尺寸

- **总宽度**: 620 + 10 + 620 + 10 + 200 = **1460px**
- **总高度**: **520px**

## 响应式行为

- 工具栏可折叠，折叠后总宽度：1250px
- 示波器可垂直滚动，支持显示更多波形
- 工具栏可垂直滚动
